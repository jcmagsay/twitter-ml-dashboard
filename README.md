## Twitter ML Dashboard

### Installation
```
yarn
```
### Development

#### Running the Dev Server

```
Save `.env.defaults` file as `.env`
```

##### Run the webpack dev server
```
`yarn start:dev`
```
##### Run the App in your browser
```
Go to http://localhost:3000/
```

### Technology

- Yarn
- Webpack 4
- React
- React Router 5
- SASS
- Britecharts React
